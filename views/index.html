API Endpoints Documentation

Create User

This endpoint is used to create a new user in the system.

Endpoint: POST /user Parameters: - name (string): The name of the user. - password (string): The user's password. - email (string): The user's email address.

Find User

This endpoint is used to retrieve information about a specific user by their ID.

Endpoint: GET /user/:id Parameters: - id (integer): The unique ID of the user to retrieve.

Find Users

This endpoint is used to retrieve a list of all users in the system.

Endpoint: GET /users

Delete User

This endpoint is used to delete a specific user by their ID.

Endpoint: DELETE /user/:id Parameters: - id (integer): The unique ID of the user to delete.

Update User

This endpoint is used to update information about a specific user by their ID.

Endpoint: PUT /user/:id Parameters: - id (integer): The unique ID of the user to update. - name (string): The new name of the user. - password (string): The new password of the user. - email (string): The new email address of the user.

Authenticate User

This endpoint is used to authenticate a user by email and password and receive a JSON Web Token (JWT) for authorization.

Endpoint: POST /user/authenticate Parameters: - email (string): The user's email address. - password (string): The user's password.
